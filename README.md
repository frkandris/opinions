# Vélemények — party játék

Magyar nyelvű, Vercelre deployolható webjáték.

## Játékmenet

- Játékosok felvétele (min. 2)
- Minden játékos megad 1 megosztó állítást
- Minden állításra minden játékos:
  - Egyetértek / Nem értek egyet
  - Tipp: ki írta az állítást
- Végén pontok és összesítés

## Fejlesztés

```bash
npm install
npm run dev
```

Ezután: http://localhost:3000

## Deploy Vercelre

A projekt Next.js, így a Vercel automatikusan felismeri.

- Build Command: `npm run build`
- Output: (alapértelmezett Next.js)
